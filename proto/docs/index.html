Component

Components are the building blocks of a Workload.

All their invocations happen over WIT Interfaces, and state is ephemeral.

While they can TCP connect outbound, they cannot accept inbound connections (fail bind call).

allowed_hosts string repeated

Allowed Hosts for TCP Outbound connections. "localhost" is always allowed. Can contain wildcards "*" as any part of the hostname.Ex: "*.wasmcloud.io" but not "som*thing.wasmcloud.io" Note this controls socket connections but not DNS lookups. Upon a successful lookup, the host will allow outbound connections to the specified hosts.

WitWorld

Represents the WIT World (WebAssembly Interface Types)

World Resolution is the intersection of all Component 'root' interfaces with Host interfaces. It must overlap 100%.

FieldTypeLabelDescription

Host interfaces that are available to the components. The 'config' on each host_interface may carry Workload Specific configuration.

Workload

Workloads are Components that when combined form a complete unit of execution (a.k.a: WIT World fully resolved).

Workloads also carry an extra Component ( Service ) that can interact with the fully resolved WIT World.

WorkloadState

NameNumberDescription
WORKLOAD_STATE_UNSPECIFIED 0

Sentinel value, should not be used
WORKLOAD_STATE_STARTING 1

Workload "start" received by the host and workload is starting. The workload_id is now in the host's internal state. The host might have not performed internal checks yet, resulting in the workload not being started and going into the "error" state.
WORKLOAD_STATE_RUNNING 2

Workload passed checks and is considered running
WORKLOAD_STATE_COMPLETED 3

Workload completed execution This indicates the workload executed to completion Useful for wasi:cli/run workloads
WORKLOAD_STATE_STOPPING 4

Workload "stop" received and stopping The host continues to report the workload status until the workload is fully stopped. Then the host removes the workload from its internal state.
WORKLOAD_STATE_ERROR 5

Workload failed to start or stopped due to an error
